# Cbilling.TV IPTV — Аддон для Kodi

[![License: AGPL-3.0-only](https://img.shields.io/badge/license-AGPL--3.0--only-blue.svg)](LICENSE.txt)
[![Kodi: 19+](https://img.shields.io/badge/kodi-19%2B-blue.svg)](https://kodi.tv/)
[![Python: 3.8+](https://img.shields.io/badge/python-3.8%2B-green.svg)](https://www.python.org/)

🇬🇧 [English version](README.md)

Видео-аддон для Kodi для просмотра IPTV-каналов и VOD-контента сервиса [Cbilling.TV](https://cbilling.tv). Требуется действующая подписка Cbilling.TV.

## Возможности

- **Live TV** — просмотр IPTV-каналов с наложением EPG
- **Архив** — доступ к записям передач (до 7 дней)
- **EPG** — электронная программа передач с локальным SQLite-кешем
- **VOD** — медиатека фильмов и сериалов с предзагрузкой метаданных
- **Избранное** — сохранение и управление любимыми каналами локально
- **Поиск** — поиск по VOD-контенту

## Требования

- **Kodi 19+** (Matrix, Nexus или Omega)
- **Python 3.8+** (входит в состав Kodi 19+)
- Активная подписка [Cbilling.TV](https://cbilling.tv)
- Public Key из личного кабинета Cbilling.TV

## Установка

1. Скачайте последний релиз ZIP со страницы [Releases](https://github.com/HiDiv/cbilling-iptv/releases) (или соберите из исходников с помощью `build_addon.py`).
2. В Kodi перейдите в **Настройки → Дополнения → Установить из файла .zip**.
3. Выберите скачанный файл `plugin.video.cbilling.iptv-<версия>.zip`.
4. Дождитесь подтверждения установки.

Подробные инструкции по установке см. в [INSTALL.md](INSTALL.md).

## Настройка

1. Перейдите в **Дополнения → Видео дополнения → Cbilling.TV IPTV**.
2. Нажмите правой кнопкой → **Настройки**.
3. Введите **Public Key** из личного кабинета Cbilling.TV.
4. (Опционально) Выберите предпочтительный сервер потоков или оставьте авто.


# Установка addon в Kodi

## Версия: 2.0.1-dev (Development)

⚠️ **Внимание:** Это dev-версия для тестирования миграции на новый API.

## Быстрая установка

### Шаг 1: Получить пакет
Файл находится в папке `dist/`:
```
plugin.video.cbilling.iptv-2.0.1-dev.zip
```

### Шаг 2: Скопировать на устройство с Kodi
- USB флешка
- Сетевая папка
- Облачное хранилище

### Шаг 3: Установить в Kodi
1. Откройте Kodi
2. Перейдите: **Настройки** → **Дополнения** → **Установить из файла .zip**
3. Найдите и выберите `plugin.video.cbilling.iptv-2.0.1-dev.zip`
4. Дождитесь сообщения об успешной установке

### Шаг 4: Настроить addon
1. Перейдите: **Дополнения** → **Видео дополнения**
2. Найдите **Cbilling.TV IPTV**
3. Нажмите правой кнопкой → **Настройки**
4. Заполните:
   - **Public Key** (Код доступа) - из личного кабинета Cbilling
   - **API URL** - адрес API сервера (уже заполнен по умолчанию)
   - **Stream Server** - выберите из списка или оставьте авто

### Шаг 5: Запустить
1. Откройте addon из меню **Видео**
2. Выберите категорию (Live TV, Архив, VOD)
3. Наслаждайтесь просмотром!

- Kodi 19+ (Matrix, Nexus, Omega)
- Активная подписка Cbilling.TV
- Public Key из личного кабинета
